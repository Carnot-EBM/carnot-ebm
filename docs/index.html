Verify LLM output with energy-based reasoning

Carnot is an open-source framework that combines Energy-Based Models with Large Language Models to detect and repair hallucinations. Built in Rust and Python/JAX for production performance and research flexibility.

The Problem
LLMs guess. They don't verify.
Autoregressive models predict the most probable next token. This produces fluent text but has no mechanism for checking logical consistency, factual accuracy, or constraint satisfaction. Errors cascade irrecoverably.
⚠️
Token-by-Token Blindness
LLMs commit to each token before seeing the rest. An incorrect early token poisons the entire sequence with no way to go back.
✅
Holistic Energy Evaluation
Energy-Based Models assign a scalar energy E(x) to complete outputs. Low energy = valid. High energy = constraint violation. Assess the whole output, then repair what's broken via gradient descent.
⚠️
No Proof of Correctness
An LLM can generate a confident, well-formatted answer that is factually wrong. There's no built-in mechanism to certify correctness.
✅
Verification Certificates
When energy reaches zero, all constraints are mathematically satisfied. Carnot produces per-constraint verification certificates that prove correctness.
Capabilities
Everything you need for energy-based verification
From low-level EBM training to high-level LLM integration, autoresearch, and GPU compute.
🧩
Three Model Tiers
Ising (lightweight, O(d²)), Gibbs (multi-layer MLP), Boltzmann (deep residual + attention). Each in Rust and Python/JAX.
🔬
Training Algorithms
Contrastive Divergence, Denoising Score Matching, Noise Contrastive Estimation, and Self-Normalised Likelihood. Adam with gradient clipping.
🔍
Verifiable Reasoning
Encode constraints as energy terms. Gradient-based repair fixes violations automatically. Verification certificates prove constraint satisfaction. SAT, graph coloring, code, Sudoku.
🤖
LLM-EBM Integration
Logprob rejection sampling (+10% accuracy), composite scoring (logprob + structural tests), iterative verify-and-repair loop for code generation.
🧠
Activation Analysis
Per-layer transformer hook extraction, hallucination direction detection (80% accuracy, 0.945 AUROC), layer-targeted EBM steering, concept vectors.
🔮
Autoresearch Pipeline
Self-improving research loop: hypothesis generation, sandboxed execution (Docker+gVisor), three-gate evaluation (energy, time, memory), automatic skill extraction.
🎮
GPU Compute
Vulkan backend via wgpu (tested on AMD Radeon 890M), distributed browser GPU compute via WebGPU gateway, ROCm support with 3.3x speedup.
🔐
Cross-Language Serialization
safetensors format enables models trained in Python/JAX to load in Rust and vice versa. Zero-copy numpy array transfer via PyO3.
⚙️
Production Quality
1,153 tests with 100% code and spec coverage. Pre-commit hooks, clippy, ruff, mypy strict. Every test traces to a spec requirement.

Quick Start
Up and running in minutes
Choose your language. Both produce equivalent results.
Python Install & train an Ising model 
# Install from source
pip install -e ".[dev]"

from carnot.models.ising import IsingModel
from carnot.training import contrastive_divergence

# Create and train
model = IsingModel(dim=64)
model = contrastive_divergence(
    model, data,
    steps=1000, lr=0.01, k=10
)

# Verify constraints
cert = model.verify(output, constraints)
assert cert.energy == 0.0  # proven correct
Rust Build & use the core library 
// Add to Cargo.toml
// carnot-ising = { path = "crates/carnot-ising" }

use carnot_ising::IsingModel;
use carnot_training::contrastive_divergence;

let model = IsingModel::new(64);
let trained = contrastive_divergence(
    &model, &data,
    1000, 0.01, 10,
);

// Save for Python interop
trained.save_safetensors("model.safetensors")?;

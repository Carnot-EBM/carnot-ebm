# Carnot Hallucination Detector

Interactive demo for scoring LLM activations with pre-trained Energy-Based Models.

Upload activation vectors (safetensors format) or paste them as JSON to get per-token hallucination energy scores.

See the [Carnot-EBM organization](https://huggingface.co/Carnot-EBM) for all models and datasets.
